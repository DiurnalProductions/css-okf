# Concepts

* [CSS Animations and Transitions](animations.md) - Time-based visual changes using transitions, keyframe animations, and motion preferences
* [CSS Box Model](box-model.md) - How margin, border, padding, and content boxes determine element size and spacing
* [The CSS Cascade](cascade.md) - The algorithm that determines which declaration wins when multiple rules apply to the same element
* [CSS Display](display.md) - How elements generate boxes and participate in block, inline, flex, or grid formatting contexts
* [Flexbox Layout Model](flexbox.md) - One-dimensional layout system for arranging elements in rows or columns
* [CSS Grid Layout](grid.md) - Two-dimensional layout system with explicit rows, columns, tracks, and named areas
* [Modern CSS](modern-css.md) - Contemporary CSS features including custom properties, math functions, nesting, layers, and logical properties
* [CSS Positioning](positioning.md) - Placing elements relative to normal flow, ancestors, or the viewport using position and inset properties
* [Responsive Web Design](responsive-design.md) - Adapting layout and typography to viewport and container size using fluid units and conditional rules
* [CSS Selectors](selectors.md) - Patterns that match elements in the DOM so declarations can be applied
* [CSS Specificity](specificity.md) - A weighting system that ranks selectors when the cascade must break ties
