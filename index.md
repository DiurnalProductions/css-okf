# CSS Knowledge Pack

A structured learning path for modern CSS — from how rules compete to how layouts adapt across viewports.

## Start here

If you are new to CSS, begin with [Selectors](/concepts/selectors.md). Selectors are how you target elements; every other concept assumes you can write a valid rule that applies to the right nodes in the DOM.

Read concepts in order the first time through. After that, use prerequisites and related links in each file to jump to what you need.

## Recommended progression

1. [Selectors](/concepts/selectors.md) — target elements with type, class, ID, attribute, and combinator selectors
2. [Cascade](/concepts/cascade.md) — how conflicting rules are resolved
3. [Specificity](/concepts/specificity.md) — the weighting system behind cascade tie-breakers
4. [Box Model](/concepts/box-model.md) — margin, border, padding, and content boxes
5. [Display](/concepts/display.md) — block, inline, flex, grid, and how elements participate in layout
6. [Positioning](/concepts/positioning.md) — static, relative, absolute, fixed, and sticky placement
7. [Flexbox](/concepts/flexbox.md) — one-dimensional layout along a main axis
8. [Grid](/concepts/grid.md) — two-dimensional tracks, lines, and areas
9. [Responsive Design](/concepts/responsive-design.md) — media queries, fluid layouts, container queries
10. [Modern CSS](/concepts/modern-css.md) — custom properties, `clamp()`, nesting, and contemporary patterns
11. [Animations](/concepts/animations.md) — transitions, keyframes, and motion performance
